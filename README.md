# 3Grecep Websites Store

Website toko online untuk menjual makanan dan snack.

## Fitur
- 🏠 Navigasi dengan dropdown menu
- 🍖 Tampilan produk dengan card layout
- 📱 Responsive design (mobile-friendly)
- 🔗 Tombol "More" untuk setiap produk
- 📞 Link WhatsApp, Instagram, Facebook

## Teknologi
- HTML5
- CSS3
- JavaScript (opsional)

## File Utama
- `Website.html` - File HTML utama
- `Wstyle.css` - Styling CSS

## Cara Menggunakan
1. Clone repository ini
2. Buka `Website.html` di browser
3. Sesuaikan link dan gambar sesuai kebutuhan
